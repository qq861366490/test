简单滚动轮播（重）

1、滚动轮播布局原理
     1、所有的图片(img)排列成一排（图片使用float）

     2、装图片的盒子(img-wrap)的宽度 = 一张图片的宽度 * 图片的个数

     3、为了能够只看到一张图片，最外面的盒子(banner)大小和图片一致，并且设置  overflow:hidden样式

2、滚动轮播js核心：
     1、使用一个变量（index）记录当前显示的图片的下标

     2、左点击  index--  （注意左边界问题）

     3、右点击  index++  （注意右边界问题）

     4、img-wrap往左移动的距离  left = 一张图片的宽度 * index * -1 (往左移动left为负)
        （比如需要显示的图片下标为3，那么就是第4张图片，自己前面就是3张图片）


无缝滚动（重）

1、无缝滚动布局原理

     在图片的最后 添加第一张图片

2、无缝滚动 js核心：
     边界处理：
         到达最后一张图片继续往左移动，需要瞬间将img-wrap的left改为0,index改为1
         到达第一张图片继续往右移动，需要瞬间将img-wrap的left改为 -1*图片的宽度*(图片的个数-1),index改为length-2

     小圆圈处理：
          当index为length-1时（切换到虚假的第一张图片）,第一个小圆圈高亮
          index==length-1 ? 0 : index
