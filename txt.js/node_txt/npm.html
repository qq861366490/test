npm install 模块名 安装包 -g : 全局安装 没有参数 : 安装到项目本地 --save : 安装到项目本地并添加记录(项目依赖) --save-dev或-D : 安装到项目本地并添加记录(开发依赖) npm install 没有任何参数 一次性安装所有配置文件记录的模块 npm install 模块名@版本号 安装指定版本的模块 npm update 没有任何参数 一次性更新所有配置文件记录的模块 npm uninstall 模块名 卸载包 -g : 卸载全局安装的模块 没有参数 : 卸载本地安装的模块 --save : 卸载本地的模块同时,删除配置文件的项目依赖(记录) --save-dev: 卸载本地模块同时,删除配置文件的开发依赖(记录) npm ls 查看已安装的模块 npm info 模块名 查看模块相关信息 (历史版本号等) ^2.1.0 主版本号保持不变 子版本号安装最新 ~2.1.0 主版本号和子版本号保持不变 修正版本号保持最新 * 安装最新版本的模块 npm cache clean 清除npm 缓存
